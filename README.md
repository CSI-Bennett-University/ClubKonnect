### Installation

1. Install dependencies
   ```sh
   pip install -r requirements.txt
   ```
2. Run migrations
   ```sh
   python manage.py migrate
   ```
3. Create a superuser
   ```sh
   python manage.py createsuperuser
   ```
4. Run the development server
   ```sh
   python manage.py runserver
   ```
